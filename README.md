# Summator

**Summator** is a simple web app that counts from 1 to a user-specified number, featuring custom CSS for a clean design.

## How to Use

1. Clone the repo:
   ```bash
   git clone https://github.com/YOUR_USERNAME/Summator.git
   ```
2. Open `index.html` in a browser.

## Tech Used

- HTML
- CSS
- JavaScript
